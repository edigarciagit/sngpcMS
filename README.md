# SNGPC Microservice (sngpcMS)

## Visão Geral
Este projeto visa implementar um pipeline de dados e serviços para integração entre o Sistema Nacional de Gerenciamento de Produtos Controlados (SNGPC) da Anvisa e sistemas internos, garantindo a conformidade regulatória (Portaria 344/98).

## Objetivos
*   **Mapeamento de Dados**: Relacionar registros de medicamentos da Anvisa com listas de controle especial.
*   **Conformidade**: Assegurar a categorização correta de medicamentos (Industrializados e Manipulados).
*   **Automação**: Monitoramento de novos registros e atualizações nas listas de controle.
